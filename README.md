# Mejor Tech LLC Website

A lightweight, responsive company website for Mejor Tech LLC.

## Run locally

```bash
npm start
```

Open `http://localhost:3000`.

## Deploy on Railway

1. Create a new Railway project from this GitHub repository.
2. Railway will detect Node.js automatically.
3. The start command is `npm start`.
4. Generate a public Railway domain after deployment.

The server automatically uses Railway's `PORT` environment variable.
